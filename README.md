# Demonstrativo Financeiro com Bootstrap, Cards e Gráficos

## Funcionalidades
- Layout responsivo com Bootstrap
- Filtros dinâmicos (Unidade, Ano, Mês, Tipo)
- Cards com totais de Débito e Crédito
- Gráfico combinado (barras + linha) usando Chart.js
- Exportação para PDF

## Como usar localmente
1. Coloque todos os arquivos na mesma pasta.
2. Abra `index.html` no navegador.

## Publicar no GitHub Pages
1. Suba todos os arquivos para o repositório.
2. Ative GitHub Pages em Configurações > Pages.
3. Acesse a URL gerada.
